# Image Classification Model

This project is made with [Teachable Machine](https://teachablemachine.withgoogle.com/train/image), a no-code tool by Google that allows you to create machine learning models directly in your browser.

## About Teachable Machine

Teachable Machine is a web-based application that makes it easy to create custom image classification models without requiring advanced machine learning knowledge. Simply upload or capture images, teach the model to recognize different categories, and export it for use in your projects.

## Project Files

- `index.html` - Web interface for the image classification model
- `model.json` - Model configuration file
- `metadata.json` - Model metadata
